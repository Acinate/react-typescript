# React Typescript
This project implements React w/ Typescript that is served by a Node.js Express application.

### Features
* Express API
* Webpack bundling
* Hot Reloading
* Typescript
* Sass components
* Environment Variables
* React Testing with Jest
* Express Testing with Jest
* Docker configuration
